# Maintainers

PatientMatch is currently maintained by:

- `@pramosferrer`

Maintainers are responsible for:

- Keeping the storefront aligned with the public serving contract.
- Reviewing changes for patient safety, privacy, accessibility, and maintainability.
- Preventing service-role database access from entering storefront runtime code.
- Removing public issue or pull request content that includes personal health information or sensitive personal data.
- Separating patient-facing storefront work from ingestion, registry parsing, and PMQ generation work that lives outside this repository.

Contributors should read `CONTRIBUTING.md`, `SECURITY.md`, and `CODE_OF_CONDUCT.md` before opening issues or pull requests.
